Application Software

Application software is a kind of software that can perform specific tasks. It is the software that helps users to perform specified Application like preparing documents, producing bills, manipulating images and videos, preparing SLC result etc. Application software are preapred on the basis of requirement of users.
The two types of application software are:
1. Packaged Software
2. Cuztomized or Tailored Software

Packaged Software: Packaged software is the application software developed by the software compaanies for all general users. They are readymade software which can perfrom tasks like preparing documents, keeping account, editing videos and images etc.
Some example of packaged software are:
1. Word Processing Package Software
2. Database Management System Software
3. Spreadsheet Software
4. Multimedia Software
5. Presentation Software
6. Graphics Software

Customized or Tailored Software: Customized software (also called tailored software) is the application software that can fulfuill requirement of a person, company or ogranization. It is designed by software companies, programmers after studying the requirement of the user, companies or organizations.
