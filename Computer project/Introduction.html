Introduction to Computer Software

A computer is made up of two parts: Computer Hardware and Computer Software.
Hardware is the physical part of the computer that takes information from software. E.g: Printer, Mouse etc.
A set of Instructions given to a computer to perform a task is known as computer program.
Computer Software is a collection of programs hardware to perform tasks. It guides computer on what to do. E.g: Linux, Windows 7 etc.

Types of Software are:
1. System Software
2. Application Software

Types of System Software are:
1. Operating System Software
2. Language Processer
3. Utility Software
